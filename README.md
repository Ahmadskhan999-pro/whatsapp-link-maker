# WhatsApp Link Maker

WhatsApp Link Maker turns any phone number and a ready message into a wa.me chat link and a matching QR code that opens WhatsApp with one tap.

## Live tool

Open it here: https://Ahmadskhan999-pro.github.io/whatsapp-link-maker

## How to use it

1. Pick your country and type your WhatsApp number.
2. Write the message you want customers to start with, or tap a ready template.
3. Copy your link or download the QR code and put it wherever your customers are.

## Good to know

The QR code downloads at print quality, so it is big enough to go on posters and shop counters.

It is free with no signup and works on any phone.
